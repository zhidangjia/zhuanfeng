        # 个人简介 - 朱岸峰

        作者：朱岸峰 (opens new window)

        本站地址：https://zhuanfeng.netlify.app/ (opens new window)

        20年+的IT领域项目及产品经验，具备从项目初期概念构思到最终交付/市场销售的全链路能力，从事过领域包括政企、运营商、军工等。

        目标导向的心智，具备丰富的解决方案能力；沟通能力强，善于向项目干系人介绍复杂业务场景，并多次担任省级项目负责人。

        优秀的学习创新及业务策划能力，多次开创从0到1的业务领域，并适应快速发展且不确定的环境。

        良好的自驱力，主动沉淀有效的工作方法，并多次荣获业内奖项。

        拥有开放合作的心态，坚信一群优秀的人可以走的更远。

        良好的身体与抗压能力，热爱中长跑等体育运动。

        # 学习经历

        初高中：就读于安徽省旌德中学。在1999年的高考中，取得县理科状元。

        大学：就读于北京大学 - 计算机科学与技术系。

        # 从业经历

        2006年起，任职于江苏省邮电规划设计院（现名中通服咨询设计研究院有限公司） (opens new window)，从事信息化领域业务开拓，直至2014年初离开。

        随后入职电信运营商体系（江苏电信） (opens new window)，从事移动互联网产品开发和运营。

        2018年，加盟北大师兄的创业公司（南京网感至察） (opens new window)，方向是NLP领域。

        2020年，入职中国电科集团子公司（CETC） (opens new window)，为国防事业贡献自己的力量。

        # 主要荣誉

        2010年，主导《数字大丰港信息化规划》，荣获江苏省优秀工程咨询成果（一等奖） (opens new window)；

        2012年11月，作为核心成员开发的《基于云服务的瑞智能耗监测管理决策分析系统》荣获南京市科学技术奖（三等奖） (opens new window)；

        2015年6月，作为核心产品负责人，荣获江苏互联网金融创新企业奖 (opens new window)；

        2016年9月，作为主创人员参与的《数据中心VR可视化管理平台》，荣获第五届中国创新创业大赛军转民大赛（三等奖） (opens new window)
