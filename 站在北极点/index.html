# 站在北极点

原文于 2009年6月26日 发表于QQ空间 (opens new window)

作者：朱岸峰

又是一个周末的下午。

或许是早上飘落几滴雨的缘故，天空格外的清澈，坐在办公室里，可以清晰的看见紫金山的天文台，感觉就像在旌中踢球时看梓山，那样触手可及。

日子就这样，悄无声息的溜走，一晃离过年都快半年。工作上，这半年似乎一无所获。前几天，一个合作伙伴说我们业绩很“干”，好贴切的字，赞一个。我只是感觉，工作好似站在了北极点，四处望去，都是北，于是也就找不到了北。

好在生活依然有很多事要去做，今天体检，我笑着说，没事，别紧张。其实，心里在默默的求包括但不限于上帝、菩萨等的保佑。这年头去体检，没几个不提心吊胆的，医院的气氛总是更适合拍恐怖片，人人似乎都被无形的手掐着脖子。也许将来我们伟大的祖国会给你更多的医保，谁知道？反正我不知道。

夏天到了，可惜我已不再暑假很多年，也不知道我《足球》系列里出现和将出现的人物，现在是否和我一样，开心时小搞点啤酒。忽然想到一个大场面，调调大家的胃口，哈哈：

旁白甲：一个酒坛高手应该具有这样的气质：一口喝下去，左手叉腰，右手平端空酒杯，面无表情。。。若是旁边有个高手把手搭在他的肩膀是，表现出英雄相吸的感觉，那就酷毙咯~。。。毫无疑问，岸子具备这样的气质，难道他就是万中无一的高手？？

旁白乙：传说中的酒坛高手在酒桌上与他人的明显区别就是，一个只喝酒不说话，一个只说话不喝酒，高手中的高手才能够达到一边喝酒一边说话的境界。比如我们的主席，整晚的喝酒……说话……说话……喝酒，乐此不疲！！！！好不快活

祝看完此文的各位胖瘦朋友们：都能找到北，并身体健康!
